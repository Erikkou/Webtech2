Cursus Bootstrap Menubalk

De menubalk wordt hier een navbar genoemd. Hier is wat je moet weten voordat je begint met de navbar:

  • Navbars vereisen een omhullende .navbar met .navbar-expand{-sm|-md|-lg|-xl} voor responsieve collapses en kleurenschema-klassen.
  • Navbars en hun inhoud zijn standaard vloeibaar. Gebruik optionele containers om hun horizontale breedte te beperken.
  • Gebruik onze spatiëring- en flex utility klassen voor het regelen van spatiëring en uitlijning binnen navbars.
  • Navbars zijn standaard responsief, maar je kunt ze eenvoudig aanpassen om dat te veranderen. Responsief gedrag hangt af van onze Collapse JavaScript plugin.
  • Navbars zijn standaard verborgen bij het afdrukken. Dwing ze af om afgedrukt te worden door .d-print toe te voegen aan de .navbar. Zie de display utility klasse.
  • Zorg voor toegankelijkheid door een <nav> element te gebruiken of, als je een meer generiek element zoals een <div> gebruikt, voeg role="navigation" toe aan elke navbar om deze expliciet te identificeren als een belangrijk gebied voor gebruikers van assistieve technologieën.

De code is te vinden op https://getbootstrap.com/docs/5.3/components/navbar/
